== Description ==

At Wikio we aim continually to provide bloggers with specially developed services for their blogs. Our latest tools allow you to display on your blog the origins of your incoming links, an automatic blogroll based on your exchanges with other bloggers, and also the tags corresponding to each of your articles.  

Let's introducing the Backlinks Widget Dashboard!

This widget allows you to display in your dashboard articles on other blogs that have cited you. It is automatic, so you need only install it and that's it.  

The simplified interface will allow you to personalize all your Wikio tools without any technical know-how at all.

The sidebar version is available : http://wordpress.org/extend/plugins/wikio-backlinks-widget/

Important: This plugin is still in BETA TESTING.


== Installation ==

1. Upload the Wikio pack into the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
